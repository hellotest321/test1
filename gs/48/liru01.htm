凌辱！女教师

第一章 阴兽学园

第一节 後视镜

一辆计程车从车站前出发，沿著街道驶向前方，愈走愈远，不知不觉中已经行驶在弯曲狭小的山路中，继续的前进中。

刚才的路虽然狭隘，却也都是舖上了沥青的平坦柏油路，而现在这条路是崎岖不平的。

从座的宫崎香澄好奇的向前张望著。

( 虽然早己听说是在山里面，可是怎麼也没想到會是在深山里面。 )

香澄的脑海里──

〃这里是一所建筑在风光明媚，一片翠绿中，且环境相常幽静的理想学校〃

浮出了这些在学校简介所出现的句子。

( 看样子这确实不是在骗人。 )

她一边眺望窗外，又再次的想著简介的内容。

   「对不起小姐，你到撒哈拉学园去做什麼？」

一搭上这计程车就说个不停的司机又开口问了。

( 是不是四十几的男人都这麼多话呢？ )

香澄有点不耐烦他的多话，但香澄按著性子：

   「哦！是这样的，我四月份开始要在撒哈拉学园教英文所以………」

   「哇！原来小姐你是英文老师啊！真看不出来！」

   「耶！我…………」

   「你长的这麼漂亮，我刚刚还以为您是电影明星，或是模特儿！」

   「啊！司机先生您真會夸人啊！」

香澄有点不好意思的回答著。

   「不……不…是真的，我说的都是真话。」

   「啊……不好意思！」

   「哪里！不过那间学校不管是校舍还是宿舍都盖的相常宏伟、豪华。」

   「哦…是吗？」

   「耶…不过你得小心，那里的孩子们可不好惹的，但是你也不必担心，我说是这样说，不过总是會有部份是好的吧！毕竟也是高中生了，况且………」

   「况且怎样！」

   「哦…我是说高中的孩子正在发育中。特别是看到一个这麼美丽的老师时，难保他们不會想入非非的。」

司机先生他边说边悄悄的看著後视镜。

香澄那看了一下前面的照後镜，可是她意外的发现自己并没有在後视镜中与司机先生打个照面。

通常搭计程车时，乘客可以从後视镜中与司机先生照面的。可是这辆车中的後视镜的角度却相常的奇怪。换句话说，後视镜中根本看不到乘客的脸，因为它是向下的，整个角度也是朝下。

( 啊！)

香澄不禁像发现了什麼的叫了一声。

今天香澄穿了一件年轻又时髦的迷你裙，而且是相当大胆的，迷你的再也不能再短的迷你裙。

这件迷你裙已致够迷你的，况且一坐下来，裙子就會往上提，看起来就快遮不住了。

所以香澄一上车便用手押压了裙子，并且把二支脚紧紧的并拢在一起。

但是经过了刚才一阵的摇晃之後，为了使身体平衡所以自然的张开了双脚。

张开的幅度不只是一点点而是整个的张开。

想到这里，香澄赶忙的把张开的双脚又并拢了起来。

( 终究是被偷看到了── )

香澄害羞的连耳朵都红了。

( 他一定是偷看了好一阵子。 )

香澄想像著刚刚的样子。

那司机恐怕是从香澄一上车就盯上她的。当他发现香澄的裙子很短时，就调整了後视镜角度。

一定是这样，错不了的。

而自己却一点都没有发现。

原来车子行驶在崎岖不平的山路上时，脚就被震开了，而司机先生那下流的眼神就一直没有离开过。

( 到底他看到了些什麼呢！ )

铁定是全被看光了。

这双自己也相当自信的大腿，充满了健康的气息与魅力，连自己都无法抵消挡这份诱惑了更何况是男人。

香澄断定司机先生已经把整个大腿都一览无遗了。因为平常时一坐下来就已经相当引人注意了，更何况今天还穿了件持别迷你的迷你裙，想然那露在外面的大腿就更吸引人呢！

而且裙子又这麼短，一看就可以看到最里面去了。

( 那麼…最里面一定也被………)

一想到这儿，坐在一角的香澄又下意识的将双腿并拢，虽然双脚早就并拢了，她还是不放心的再做了一次。

立刻停止想像，并一再的安慰自己，什麼也没被看到，以寻求安心。

尽管这样的自我安慰著，可是香澄不明白，为什麼老是想著那好色的司机先生已经看到了她的粉红色内裤。

这件有蕾丝的内裤是香澄最喜欢的一件。

内裤上面虽然也穿了丝袜，可是那种透明丝袜却不管用，所以透过半透明的内裤，那一丛黑色的嫩草就更是令人触目惊心的。

( 说不定也许他真的已经看到那一团──一定的，瞧他那贼溜溜的眼睛，他一定已经看到了。 )

这一那，香澄的脸像被火燃烧一般的热了起来，心跳也不由得加速。

跟平常一般，每次只要一想到什麼令人害羞的事情时候，香澄的体内便會交织著，混合有羞耻的感情，彷佛又像正在做著那件事时的甘美欢愉的感情，就这样不停的交织扩散在她的体内。

即使这只是想而已，可是只要一旦让它浮现在脑海中，它就會慢慢的在香澄的脑子里不断的扩散。

( 他！那个色眯眯的司机，一定是看见了我的内裤，而且现在也一定正在想著一些下流的事情。说不定他的裤子前面的拉链也早己拉了下来，并且正想像著我那里的颜色、形状甚至大小………或者他正想著要怎样才能跟我睡…… )

   「小姐……」

( 我就知道……他把我带到这人烟稀少的山区是有预谋的，这种事我是绝对不允许的── )

   「小姐……小姐……」

   「啊…什……什麼事……」

香澄不安的看著司机先生。

   「到了哟！」

   「哦！」

看了看窗外，左手边的石造建筑物，这不就是撒哈拉学园的大门吗？

香澄立刻回过神来，对司机先生不好意思的笑了笑，赶忙从钱包里掏出钱来付车资。

下了计程车踩在真实的土地上才有安全的感觉。

( 哇…终於到了……真好……… )

有了安全感也就迅速的忘了刚才的胡思乱想。

这所建筑在山丘上的新校舍，被重重的山峦及林木所围绕著，此刻刚好映在夕阳西下的晚霞中，那景色有如图画一般的美丽。

校舍里面还包含了一座建在半山腰中，可容纳各年级各四个班的全体学员及单身教师的大宿舍。

尽管这无疑的是一所华美的校舍，然而香澄却有著强烈的不适应感。

丑闻 恶势力猖獗 在泥泞地里喘息 人善被犬欺 黑色的炮弹 甜蜜的恶梦 美妙的喷火口 完美的结局
